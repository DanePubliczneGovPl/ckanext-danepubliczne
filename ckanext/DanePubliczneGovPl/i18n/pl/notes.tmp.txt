Specify data reuse conditions
Określenie warunków ponownego wykorzystywania


We publish public data so they can be reused freely and unrestricted.
      There are exceptions from this rule mentioned below, but we encourage not to restrict them if there is no clear case for that.
Informacje publiczne są udostępniane w celu ich ponownego wykorzystywania bez ograniczeń warunkami i
      bezpłatnie.
      Istnieją od tej zasady wymienione poniżej wyjątki, jednak zachęcamy, aby ich nie stosować jeżeli nie
      przemawiają za tym mocne racje.

One should inform about the source of this data
Odbiorca powinien poinformować o źródle pozyskania informacji

One should inform about the creation time of reused information as well as time of access to it
Odbiorca powinien poinformować o czasie wytworzenia i pozyskania informacji od dostawcy

One should publish a copy of originally accessed data when publishing reused information
Odbiorca powinien zawrzeć kopię informacji w pierwotnie pozyskanej formie udostępniając przetworzoną informację

One should inform about modifications to originally accessed information
Odbiorca powinien poinformować o przetworzeniu informacji, gdy ją jakkolwiek modyfikuje
